# 2.1 Values and types
+ value: a letter or a number
+ types
   + **integer**: type ```int```; 2
   + **string**: type ```str```; enclosed in quotation marks.
      * "Hello, World!"; "17"; "3.2"
   + **floating point**: type ```float```; 3.2
   
   ----
   
   ```
   # print() for integers
   >>> print(4)
   4
   
   # print() for strings
   >>> print("Hello, World!")
   Hello, World!
   >>> Print('Hello, World!')
   Hello, World!
   ```
   
   ```
   # type() can tell you what type a value has
   >>> type(17)
   <class 'int'>
   >>> type(3.2) 
   <class 'float'>
   >>> type('Hello, World!') 
   <class 'str'>
   >>> type('17') 
   <class 'str'> 
   >>> type('3.2') 
   <class 'str'>
   ```
